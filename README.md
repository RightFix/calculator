# Calculator

A modern calculator web app with basic arithmetic, trigonometric functions, and unit converters.

## Features

- **Basic Calculator**: Addition, subtraction, multiplication, division
- **Advanced Operations**: Square (X²), Square Root (√x), Cube (X³), Cube Root (3√x), Percentage (%), Factorial (!)
- **Trigonometric Functions**: sin, cos, tan, and their inverses (sin⁻¹, cos⁻¹, tan⁻¹)
- **Unit Converters**:
  - Temperature (Celsius, Fahrenheit, Kelvin)
  - Length (mm, cm, dm, m, km)
  - Time (seconds, minutes, hours, days, weeks, months, years)
- **Keyboard Support**: Type numbers and use Enter for equals, Backspace to delete, Escape to clear

## Usage

Open `index.html` in a web browser.

### Keyboard Shortcuts
| Key | Action |
|-----|--------|
| 0-9 | Enter numbers |
| + - * / | Operators |
| ( ) | Parentheses |
| Enter or = | Calculate |
| Backspace | Delete last character |
| Escape | Clear all |

## Tech Stack

- HTML5
- CSS3 (responsive design)
- JavaScript (vanilla)
- jQuery (for animations)
